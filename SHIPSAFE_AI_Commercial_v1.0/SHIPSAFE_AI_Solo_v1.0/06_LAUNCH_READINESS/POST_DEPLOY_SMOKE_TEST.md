# Post-Deploy Smoke Test

- [ ] Correct release/commit is deployed.
- [ ] Domain and TLS are valid.
- [ ] Public and protected routes behave correctly.
- [ ] Registration/login/session behavior works.
- [ ] Primary user journey completes with real test data.
- [ ] Database reads/writes and access policies behave correctly.
- [ ] Storage/upload/download behavior works where applicable.
- [ ] Webhooks, queues, scheduled jobs, email, and external integrations work.
- [ ] No secret, stack trace, debug output, or sensitive log entry is exposed.
- [ ] Monitoring receives expected health and error signals.
- [ ] Rollback target remains available.
