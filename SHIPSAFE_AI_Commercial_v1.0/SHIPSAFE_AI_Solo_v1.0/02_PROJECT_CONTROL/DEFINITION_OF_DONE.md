# Definition of Done

A change is complete only when:

- [ ] Acceptance criteria are tested and evidenced.
- [ ] Focused tests pass.
- [ ] Relevant regression tests pass.
- [ ] Lint/type/build checks pass where applicable.
- [ ] Authentication, authorization, data, secrets, and input risks were considered.
- [ ] The diff contains no unexplained unrelated changes.
- [ ] No test, validation, security control, or error path was weakened to obtain a pass.
- [ ] Documentation and configuration examples are current.
- [ ] Rollback instructions are present.
- [ ] Remaining risk is disclosed and accepted.
