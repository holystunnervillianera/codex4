# SHIPSAFE AI v1.0

Production guardrails, verification controls, security checks, and recovery procedures for AI-built applications.

## Start here

1. Open `00_START_HERE/SHIPSAFE_AI_Quick_Start.pdf`.
2. Copy the appropriate instruction file from `01_AGENT_GUARDRAILS` into your project.
3. Complete `02_PROJECT_CONTROL/PROJECT_CONTEXT.md` and `SCOPE_LOCK.md`.
4. Use `06_LAUNCH_READINESS/SHIPSAFE_AI_Launch_Readiness_Scorecard.xlsx` to assess the project.
5. Do not launch while any Critical control is failed or unknown.

## Operating rule

A coding agent's statement that work is complete is not evidence. Completion requires independently reviewable proof: changed files, commands run, test output, security checks, and acceptance-criteria results.

## Important limitation

SHIPSAFE AI is an engineering control framework, not a security guarantee, legal opinion, compliance certification, penetration test, or substitute for qualified review of high-risk systems.
